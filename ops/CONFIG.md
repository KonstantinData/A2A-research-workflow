# Configuration

Document environment variables and secrets required for the workflow.
See repository README for common variables.

## Email
- `MAIL_FROM`
- `MAIL_USER`
- `MAIL_SMTP_HOST`
- `MAIL_SMTP_PORT`
- `MAIL_SMTP_SECURE`
- `MAIL_SMTP_PASS`

## Trigger Words
- `TRIGGER_WORDS_FILE` (defaults to `config/trigger_words.txt`)
